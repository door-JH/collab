## 설치 및 실행 방법 ⚙️

1. 프로젝트 클론
    ```bash
    git clone https://github.com/your-repository/project-name.git
    ```

2. 의존성 설치
    ```bash
    ./gradlew build
    ```

3. 실행
    ```bash
    ./gradlew bootRun
    ```
